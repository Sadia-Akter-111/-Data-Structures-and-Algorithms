# Singly Linked List in C

A simple and well-structured implementation of a Singly Linked List in C. This project demonstrates the fundamental linked list operations using dynamic memory allocation and pointers.

---

## Features

- Create a new node
- Insert at the beginning
- Insert at the end
- Delete from the beginning
- Delete from the end
- Search for a node
- Count total nodes
- Display the linked list
- Free all allocated memory

---

## Technologies Used

- C Programming Language
- Standard C Library
  - stdio.h
  - stdlib.h
