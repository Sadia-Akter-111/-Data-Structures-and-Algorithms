# Binary Search Applications in C

## Overview
This project demonstrates different applications of the Binary Search algorithm using the C programming language. Instead of searching for an element in an array, Binary Search is used to solve mathematical problems efficiently.

## Implementations

### 1. Solving a Linear Equation
Find the value of x for the equation:

> 3x + 5 = 65

Binary Search is used to determine the correct value of x by repeatedly reducing the search space.

### 2. Square Root
Find the square root of a given number using Binary Search without relying on built-in mathematical functions.

### 3. Cube Root
Find the cube root of a given number using Binary Search with high precision.

## Features
- Solve a linear equation using Binary Search
- Find the square root of a number
- Find the cube root of a number
- Efficient search with logarithmic time complexity
- Simple and easy-to-understand C implementation

## Algorithm
1. Define the search range.
2. Calculate the middle value.
3. Compare the result with the target value.
4. Narrow the search range based on the comparison.
5. Repeat until the desired answer is found.

## Time Complexity

| Problem | Time Complexity |
|---------|-----------------|
| Linear Equation | O(log n) |
| Square Root | O(log n) |
| Cube Root | O(log n) |
