# Recursion in C (Factorial, Sum, and Fibonacci)

## Overview
This project demonstrates the use of Recursion in the C programming language by implementing three fundamental problems: Factorial, Sum of Natural Numbers, and Fibonacci Series. These examples help in understanding how recursive function calls work.

## Implementations

### 1. Factorial Using Recursion
Calculates the factorial of a given positive integer using a recursive function.

### 2. Sum of Natural Numbers Using Recursion
Finds the sum of the first n natural numbers recursively.

### 3. Fibonacci Series Using Recursion
Generates the Fibonacci sequence using recursive function calls.

## Features
- Recursive implementation of Factorial
- Recursive calculation of the Sum of Natural Numbers
- Recursive generation of the Fibonacci Series
- Simple and easy-to-understand C programs

## Algorithm
1. Define a recursive function.
2. Specify the base case.
3. Call the function recursively with a smaller input.
4. Continue until the base case is reached.
5. Return the final result.

## Time Complexity

| Problem | Time Complexity |
|---------|-----------------|
| Factorial | O(n) |
| Sum of Natural Numbers | O(n) |
| Fibonacci | O(2ⁿ) |
