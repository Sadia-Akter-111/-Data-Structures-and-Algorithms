# Average Calculation Using Dynamic Memory Allocation in C

## Overview
This project demonstrates how to calculate the average of a set of numbers using Dynamic Memory Allocation in the C programming language. Memory is allocated at runtime using malloc(), allowing the program to handle different input sizes efficiently.

## Features
- Take the number of elements as user input
- Dynamically allocate memory using malloc()
- Input array elements
- Calculate the sum of all elements
- Find and display the average
- Release allocated memory using free()

## Algorithm
1. Read the number of elements.
2. Allocate memory dynamically using malloc().
3. Input the elements.
4. Calculate the sum of all elements.
5. Compute the average using:
   - Average = Sum / Number of Elements
6. Display the result.
7. Free the allocated memory.

## Time Complexity

| Operation | Time Complexity |
|-----------|-----------------|
| Input Elements | O(n) |
| Calculate Sum | O(n) |
| Calculate Average | O(1) |
| Overall | O(n) |
