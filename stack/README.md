# Stack Implementation (Static) in C

## Overview
This project demonstrates the implementation of a Static Stack using an array in the C programming language. The stack follows the Last In, First Out (LIFO) principle, where the last inserted element is the first one to be removed.

## Features
- Create a static stack
- Push elements into the stack
- Pop elements from the stack
- Peek the top element
- Check if the stack is empty
- Check if the stack is full
- Display all stack elements

## Technologies Used
- C Programming
- Array Data Structure

## Stack Operations
- Push – Inserts an element into the stack.
- Pop – Removes the top element from the stack.
- Peek – Returns the top element without removing it.
- isEmpty – Checks whether the stack is empty.
- isFull – Checks whether the stack is full.
- Display – Prints all elements of the stack.

## Time Complexity

| Operation | Time Complexity |
|-----------|-----------------|
| Push      | O(1) |
| Pop       | O(1) |
| Peek      | O(1) |
| isEmpty   | O(1) |
| isFull    | O(1) |
| Display   | O(n) |

## Learning Outcome
This project helps in understanding:
- Stack data structure
